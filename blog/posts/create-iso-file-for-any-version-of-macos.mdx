# Create an ISO File For Any Release of macOS

![Image](/img/blog/how-to-install-macos-big-sur-11-5-in-oracle-vm-virtualbox-windows.jpg)

**Today we'll guide you through the setup of creating an ISO File for any macOS Release.**

> What is it needed for?
> > macOS ISO is required for Installing macOS as an [Virtual Machine](https://en.wikipedia.org/wiki/Virtual_machine) on Windows. We have already created a step by step guide on how to [install macOS on VirtualBox](/search?q=Install+macOS). You can refer to that after reading this guide.

<!-- truncate -->

Below are the options for creating an ISO File. Select the macOS release name you want to install and you'll be guided for the rest of the process.

:::caution Important!

**macOS Mojave and Releases Older than this have been marked as deprecated by Apple, no longer supported, and no longer recieving updates. Our Recommendation is to use the latest release of macOS. However, If you want, macOS Mojave Guide is provided.**

:::
