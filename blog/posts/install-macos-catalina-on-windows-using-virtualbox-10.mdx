# How to Install macOS Catalina 10.15 on Oracle VM VirtualBox (Windows)

**Today we'll guide you how to install macOS Catalina 10.15 in Oracle VM VirtualBox.**

{/* truncate */}

**Prerequisites (Download these files by clicking on Click Here!):**

- macOS Catalina ISO File- Click Here!
- VirtualBox - Click Here! (Click on Windows hosts after clicking this link.)
- VirtualBox Extension Package- Click Here! (Click on All supported platforms under Virtualbox x.x.xx Oracle VM VirtualBox Extension Pack
- 4 GB RAM and 200GB Storage for the VM. Should be double the values for Windows System Example: You have to be with 8 GB RAM and Storage more than 500 GB. 4 GB RAM and 200GB Storage for macOS.
